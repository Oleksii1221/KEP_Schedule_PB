# KEP Schedule Public

Публічний канал встановлення та оновлень для KEP Schedule.

- `index.html` - сторінка для користувачів.
- `latest.json` - master manifest, який читає Android-додаток.
- `latest-dev.json` - dev manifest для перевірки нової Android-збірки.
- `downloads/android/master/KEP_Schedule.apk` - актуальний master APK.
- `downloads/android/dev/KEP_Schedule-dev.apk` - dev APK.
- `downloads/ios/dev/` - окреме місце для майбутньої iOS dev-версії.

Код додатка тут не зберігається.
